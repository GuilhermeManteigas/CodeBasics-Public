Language B


Due to the time constraints of this experiment we will give a brief explanation on the language B, you don't need to understand everything or memorise it but try to look at the examples and see what they each do.


Writing to the Screen:

In this language, to write information to the screen, you will have to use the command "console.log();".
Looking at the example below, what you want to be written on the screen needs to be written inside of the parenthesis. If this information is text, it needs to be in between two apostrophes. If you wish to write a number, there is no need for an apostrophe.
Example:
Code:
console.log('Hello my friend');
console.log(2023);
Output:
Hello my friend
2023


Variables and calculations:

A variable is a place where you can store data. In this language, to create a variable, you need to write "number = X ;", where X is the value you want to write. This will store the desired value in the variable number.
To perform calculations, you can do it as if you were using a calculator. For example, you could write "5 + 5", this will give the result of 10 back. You can also do calculations with variables you created previously. For example, we could do "5 + 5 + number", where number is the variable.
Example:
Code:
x = 5;
result = 10 + x;
console.log(result);
Output:
15
 
 


Conditional statements:

In this programming language, you can use conditional statements by inputting "if" and "else". These statements allow you to write conditions on your programme. For example, a statement could be made that if a variable is bigger than 10, the programme must write a message on the screen. If the number is below 10, then a different message would be written. This is useful for building logic into your programme.
These statements come in the form of "if(CONDITION){CODE}else{CODE}".
Example:
Code:
x = 10;
if(x > 5){
console.log("x is bigger than 5");
} else {
console.log("x is smaller than 5");
}
Output:
x is bigger than 5
 
 
 
 
 


Functions:

In this programming language, you can use functions. These are sub programmes inside your programme that can be called when needed. They can be used to reduce the number of times you need to write code, by allowing you to write it once and calling it at a later stage.
Functions are written in the form pf "FunctionName(parameters){CODE}". And they can later be called by writing "FunctionName(parameters);"
It is important to note that parameters are optional and can be left black if they are not needed.
Example:
Code:
function function_square(number){
console.log(number * number);
}
 
function_square(10);
function_square(5);
Output:
100
25
 
 
 
 


Loops:

In this programming language, you can use loops. These are pieces of code that will run until a condition is met. They can be used to make repetitive tasks easier.
We will be using "for" loops these are written in the form of "for(variable;condition;increment){CODE}".
Example:
Code:
for(variable = 0; variable < 3; variable = variable + 1){
console.log(variable);
}
Output:
0
1
2


Arrays:

In this programming language, you can use arrays. These allow you to store multiple numbers in one variable.

Declaring an array:
Example:
Code:
array = [1,2,3,4,5];
console.log(array);
Output:
[1,2,3,4,5]

Adding a number to an array:
Example:
Code:
array = [1,2,3,4,5];
array.push(6);
console.log(array);
Output:
[1,2,3,4,5,6]

Remove a number from an array:
Example:
Code:
array = [1,2,3,4,5];
array.splice(array.indexOf(3), 1);
console.log(array);
Output:
[1,2,4,5]

Sort an array:
Example:
Code:
array = [4,3,2,1,5];
array.sort();
console.log(array);
Output:
[1,2,3,4,5]
